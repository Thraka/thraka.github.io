Type Name Description
int width

How wide the window is in font cells.
int height

How tall the window is in font cells.
bool zoom

When true, creates a larger window to automatically zoom the contents.
Remarks

The width and height values are used to generate the game window based on how large the font is. For example, if the font is 10x8, and you request window size of 20x20, a window is created with a width of 10 * 20 and a height of 8 * 20 in pixels, which is 200x160.

When zoom is set to true, the window will try to get as big as possible while keeping the SadConsole output crisp.

SetWindowSizeInPixels(int, int)
