Remarks

When overriding this method, to perhaps restrict the requested size, make sure to do the following:

  • Check if the CanResize property allows resizing. If it doesn't, throw an InvalidOperationException.
  • The SadConsole.UI.Controls.ControlBase._width and SadConsole.UI.Controls.ControlBase._height variables of the control are set to the values provided.
  • The MouseArea property is set to the same width and height as values provided. Generally the theme will adjust this if needed, but it might ignore this so you'll need to set it.
  • The Theme, if set, is reattached by calling Attached(ControlBase).
  • Set IsDirty to true.
